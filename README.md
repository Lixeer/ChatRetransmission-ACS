## 食用方法
S-main.py文件在服务端部署，默认端口：5701（一般不会冲突，冲突修改就行）。

client.py文件在本地可以直接使用，但是需要配置好服务端。在url中填入正确的服务端域名或者IP。

服务端建议选择国外vps的最低配置即可。
